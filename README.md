# ⚡ React + Vite Portfolio — Powered by Caffeine and Curiosity

Welcome to my personal developer portfolio, built with **React** and **Vite** because I like my dev stack like I like my brain: fast, flexible, and slightly overclocked.

## 🎯 What This Is

This site showcases **my journey from fishmonger to fullstack developer** — with real projects, real code, and very real hyperfixation. It’s where I’ve collected the best of what I’ve built so far, including:

- 🔹 Interactive UI components with smooth UX and snappy feedback
- 🔹 Fullstack project highlights (school, personal experiments, and late-night coding marathons)
- 🔹 Clean, reusable code that didn’t _used_ to be clean, but I’ve evolved
- 🔹 Subtle animations because static pages are emotionally vacant
- 🔹 A lowkey obsession with performance and responsiveness

## 🧰 Tech Stack

- ⚛️ React (component wizardry)
- ⚡ Vite (dev server on rocket fuel)
- 🎨 CSS (a cocktail of custom and utility styles)
- 🧪 Deployed on Netlify, because I hate FTP

## 🐟 Origin Story

Before I learned to code, I was a fishmonger.  
Then I accidentally discovered a billing exploit in Shopify’s delivery system while updating our shop’s online store. I had no tech skills — just a sense that something was wrong and a habit of overanalyzing things.

That’s when I realized:  
**I’m supposed to build things. And break things. Then build better things.**
